# Eyrie Tools

Eyrie Tools is a collection of practical cybersecurity research utilities designed to support ethical testing, security validation, and defensive workflows.

The goal is to help security practitioners identify weaknesses responsibly and contribute to safer systems.

## Documentation

Project documentation is available at:

- [Eyrie Docs - Tools](https://romaincuchet.github.io/eyrieDocs/docs/category/tools)

## Available Tools in This Workspace

- [Supabase Endpoint Scanner](supabase/README.md): Scans Supabase REST/OpenAPI endpoints and exports structured results.

## Responsible Use

- Use these tools only on systems you own or are explicitly authorized to test.
- Follow applicable laws, contracts, and disclosure policies.
- Handle credentials and scan results as sensitive data.
